Introduction



spsim is a small program which aims to realistically simulate the output of a single particle diffraction experiment. The program reads it's input from a single file named spsim.conf which must reside on the directory where the program is run. It then runs for while and produces several VTK files related to the detector output from the simualted an experiment.
